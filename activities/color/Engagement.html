What is the difference between an ID and a Class?

ID - you can use only once and never repeat.

Class - name more that 1 tag a tag can have more than 1 class.

What is the difference between a SPAN tag and a DIV tag?

A div element is used for block-level organization and styling of page elements.

a span element is used for inline organization and styling.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

The div, header, and footer all work the same way, but what distinguishes them is the name.

The div element is by default a block element, meaning that it takes all available width, and comes with line breaks before and after.

The header element represents a container for introductory content or a set of navigational links.

main tag holds the primary content unique to the page.

The footer element defines a footer for a document or section.

What are the different ways to select a color?

There are couple ways to select colors:

  1. Named Colors
  2. Hexadecimal Color Codes
  3. RGB
  4. HSL (Hue, Saturation, Lightness)
  5. Short code
