  • The form tag has an "action" attribute. What does it do?

    The action attribute in the form tag specifies the URL where the form's data should be sent when the form is submitted.

  • The form tag has a "method" attribute. What does it do?

    The method attribute in the form tag specifies the HTTP method used to send the form data to the server.

  • What is the difference between a "POST" and a "GET"?

    If I use POST on the hmtl, other people can't see what i input into form from URL. In contrast, if I use GET, anyone can see what I input into form through URL.
