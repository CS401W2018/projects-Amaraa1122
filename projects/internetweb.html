XHTML

Definition of XHTML

XHTML (Extensible Hypertext Markup Language) is a web coding standard that is essentially a stricter version of HTML. It combines the flexibility of HTML with the rigor of XML, meaning it requires more precise syntax rules and is more strict about how elements are structed and closed.
